Welcome to Geoweaver

Geoweaver is a online software which allows users to compose and execute full-stack deep learning workflows which uses open source deep learning libraries. It provides multiple modules like Jupyter Notebook Server, Jupyter Hub, Jupyter Lab, Google Earth Engine, Process and Workflows. It also includes FTP client, Shell scripts and SSH for running system-level programs. Geoweaver allows every user to record history and manage, share, track, replicate and reuse complicated scientific workflows.

SSH, or Secure Shell, is a remote administration protocol that allows users to control and modify their remote servers over the Internet. The service was created as a secure replacement for the unencrypted Telnet and uses cryptographic techniques to ensure that all communication to and from the remote server happens in an encrypted manner. It provides a mechanism for authenticating a remote user, transferring inputs from the client to the host, and relaying the output back to the client.
